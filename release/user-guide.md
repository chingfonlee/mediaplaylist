# 媒體啟動器 — 使用說明

## 安裝

1. 雙擊 `MediaLauncherSetup.exe`
2. 點「下一步」完成安裝（**不需要管理員密碼**）
3. 安裝完成後可直接啟動，或稍後從桌面 / 開始功能表開啟

---

## 啟動程式

雙擊桌面上的「媒體啟動器」捷徑，或從開始功能表選擇「媒體啟動器」。

程式啟動後會出現一個黑色的命令視窗，接著瀏覽器自動開啟 `http://localhost:8765`，這是正常現象。

> **請勿關閉黑色命令視窗**，它是程式的核心，關掉後媒體啟動器就會停止運作。  
> 若要關閉程式，請關閉黑色命令視窗即可。

---

## 第一次加入媒體資料夾

1. 點畫面上方的「**📁 資料夾**」按鈕，展開資料夾管理列
2. 點「**＋ 加入資料夾**」→ 在對話框中選擇你的媒體資料夾
3. 點「**⟳ 重新整理**」掃描資料夾內容
4. 媒體卡片會出現在畫面中央

你可以加入多個資料夾；每個資料夾可個別勾選或取消勾選。

---

## 播放 / 開啟檔案

**單擊**任一媒體卡片，即可用電腦上對應的程式開啟：

| 類型 | 開啟方式 |
|------|---------|
| PPT / PPTX | Microsoft PowerPoint（或相容程式） |
| PDF | Edge、Adobe Reader 等 PDF 閱讀器 |
| 圖片 | Windows 相片、小畫家等 |
| 影片 | VLC、Windows 媒體播放器等 |
| 音樂 | Groove 音樂、VLC 等 |

點擊卡片右下角的「**📂**」圖示，可直接在檔案總管中定位該檔案。

---

## 搜尋與篩選

- **搜尋列**（畫面頂部）：輸入檔名關鍵字即時過濾
- **類型頁籤**：點「全部 / 影片 / PDF / 簡報」篩選特定類型

---

## 建立任務播放清單

1. 點畫面右上角的「**📋 任務**」按鈕，展開右側任務區
2. 點「**＋**」新增任務，輸入清單名稱（例如「上午課程」）
3. 從左側媒體卡片**拖曳**到右側任務區，放開後即加入

可以建立多個任務，從頂部下拉選單切換。

---

## 拖曳排序

在任務清單中，長按任一項目後**上下拖曳**即可調整播放順序。

---

## 辨識目前播放項目

任務清單中：

- **藍色左邊框 ＋ 「目前」badge**：正在播放（最後點擊開啟）的項目
- **淡藍色左邊框 ＋ 「下一個」badge**：下一個會播放的項目

點「**▶ 播放目前**」按鈕可重新開啟目前項目；  
點「**▶▶ 下一個**」按鈕會開啟下一個並移動「目前」標記。  
播到最後一個後，繼續點「下一個」會循環回第一個。

---

## 調整任務區寬度

將滑鼠移到右側任務區的**左邊界**，游標變成左右箭頭後，拖曳即可調整寬度。  
雙擊左邊界可恢復預設寬度。

---

## 常見問題

### 點了 PPT 沒有反應 / 出現錯誤

確認電腦已安裝 Microsoft Office 或 LibreOffice。  
若已安裝但仍無法開啟，請在 Windows 「設定」→「應用程式」→「預設應用程式」中，  
將 `.pptx` 與 `.ppt` 設定為 PowerPoint 開啟。

### 影片卡片沒有縮圖

影片縮圖需要 ffmpeg。若安裝包未包含 ffmpeg，影片卡片會顯示 🎬 圖示——這不影響播放，直接點擊即可開啟。

### 找不到媒體資料夾 / 掃描後沒有檔案

- 確認已在「📁 資料夾」區勾選了資料夾
- 點「⟳ 重新整理」重新掃描
- 確認資料夾內有媒體檔案（支援格式：mp4、mkv、avi、mp3、wav、pdf、pptx、ppt、jpg、png 等）

### 換電腦如何帶走設定與任務清單

1. 在舊電腦上，用檔案總管開啟：`%APPDATA%\MediaLauncher`  
   （在網址列貼上此路徑後按 Enter）
2. 複製 `config.json` 和 `tasks.json` 到 USB
3. 在新電腦安裝媒體啟動器，啟動一次
4. 關閉程式，將 USB 上的兩個檔案貼上覆蓋 `%APPDATA%\MediaLauncher\`
5. 重新啟動程式即還原

### 如何清除所有設定重新開始

1. 關閉媒體啟動器（關閉黑色命令視窗）
2. 按 `Win + R`，輸入 `%APPDATA%` 後按 Enter
3. 刪除 `MediaLauncher` 資料夾
4. 重新啟動程式，即回到初始空白狀態

> ⚠️ 此操作無法復原，任務清單與設定將永久刪除。

### 如何解除安裝

`Windows 設定` → `應用程式` → 搜尋「媒體啟動器」→「解除安裝」

解除安裝**不會刪除**你的任務清單與設定（保存在 `%APPDATA%\MediaLauncher\`）。  
若同時想清除資料，請先按上方「清除所有設定」的步驟操作。
